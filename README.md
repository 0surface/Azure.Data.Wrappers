## Azure Storage Simplified
- Queues/Blobs/Tables/File Shares
- Azure Storage Resources
- Dependancy Injection
- Mockable for testing
- Prefer async calls
- Plugs into the [King.Service](https://github.com/jefking/King.Service) task framework
